0. HTML, CSS, Javascript
	HTML = 뼈대
	CSS = 디자인 요소
	Javascript = 동작을 추가하는 것
1) HTML
	웹 문서의 구조를 정의하고 콘텐츠를 표현하는 기본 마크업 언어

2) CSS(Cascading Style Sheet)
	html문서에 레이아웃과 디자인을 포함한 시각적 요소를 정의하기 위한 규격
	html은 콘텐츠 내용과 구조를 정의하고 css에서는 콘텐츠의 위치, 정렬, 글자크기, 여백, 배경색상 등을 정의
	디작인을 적용할 html 요소를 셀렉터(selector)로 지정해 원하는 디자인 속성을 부여

3) Javascript
	html 문서에 이벤트 처리나 동적으로 변화하는 콘텐츠를 표현하거나 서버와의 연결을 통해 데이터를 가지고 오는 등의
	프로그램적인 요소
	html은 텍스트 파일로 한 번 작성된 내용은 수정하기 전까지 항상 내용이 동일
	서버에 새로 접속하지 않고도 화면의 정보가 변하는 기능
	자바스크립트는 프로그래밍언어로 html문서를 동적으로 처리하고 이벤트(버튼 클릭 등)를 처리할 수 있도록 하는 기술

1. HTML(HyperText Markup Language)
	HyperText => HyperLink를 사용할 수 있는 텍스트 => 문서를 서로 연결해주는 링크를 의미
	Markup => 문서에 표시하는 것을 의미 => 표시하다라는 뜻
	웹 문서를 만드는 언어
	
	HTML5를 HTML로 부르게 되었을까?
	- 2014년 11월에 발표된 HTML 표준 규약은 이전 버전과 구별하기 위해 HTML5로 불렀다
	- 다양한 기능이 추가되었기 때문에 HTML5로 불럿으나 최근 대부분의 웹 브라우저에서 HTML5를 지원하게
	되면서 HTML5에서 숫자 5를 빼고 그냥 HTML이라고 부른다

1) HTML 구조
	기본구조는 <!DOCTYPE html>로 시작해서 <html>, <head>, <body> 3개 영역으로 구성됨
	<!DOCTYPE html>		: 현재 문서가 HTML5 언어로 작성된 웹 문서라는 의미
	<html>			: <html>~</html> 웹문서의 시작과 끝을 나타내는 태그
		<head></head>	: <head>~</head> 웹 브라우저가 웹 문서를 해석하는데 필요한 정보를
					입력하는 부분
		<body></body>	: <body>~</body> 실제로 웹 브라우저 화면에 나타나는 내용
	</html>

2) 태그의 기본 구조
	<시작태그 속성1="속성값1" 속성2="속성값2">내용</종료태그>
	
(1) 태그(tag)
	<>로 표시한다
	일반적으로 <시작태그></종료태그> 한쌍으로 사용한다
	시작태그만 있는 것들도 존재하며 빈태그라고 부른다 ex) <hr />, <br />

(2) 내용(content)
	시작태그와 종료태그 사이에 있는 내용

(3) 엘리먼트(element)
	태그 + 내용

(4) 속성(attribute)과 속성값(value)
	태그<>안에 있는 일종의 옵션(설정)
	속성="속성값"으로 사용
	여러 속성을 지정할 경우 각 속성을 띄어쓰기로 구분한다
	ex) <h1 align="right" id="ex">내용이다</h1>

3) 태그의 기본 구조2
- html 태그
	태그 <>를 사용하여 나타낸다
	종료 태그 앞에 </>가 붙게 된다
	태그 안에 또 다른 태그가 들어갈 수 있으나 열어준 순서대로 닫아야 한다
	일부태그는 닫는 태그가 없는 경우도 있다
	html의 주석은 <!-- -->로 표시한다

- 속성(attribute)
	html 태그에 부가적인 정보를 제공하는 방법
	그림을 넣기 위해 사용하는 <img> 태그의 경우 그림파일의 위치를 지정하기 위한 src 속성이 있다
	태그에 따라 지원하는 속성이 다르며 여러 속성을 나열해 지정하는 것이 가능하다
	항상 시작태그에 작성하며 속성명=값 형태로 구성된다
	html5에서는 디자인적인 표현을 위한 속성들이 css로 이전되어 태그에서 사용할 수 있는 속성은 그리 많지않다

- 태그바디(Tag Body)
	태그와 태그 사이에 콘텐츠를 의미한다

3. 태그정리
1) <head> 태그 내부에 들어가는 태그

- <title>
	<title></title>	웹페이지의 제목을 나타내는 태그
- <meta>
	<meta /> 웹페이지의 보이지 않는 정보를 제공하는데 쓰이는 태그
	페이지의 설명요약, 핵심키워드, 제작자, 크롤링 정책 등 많은 정보 제공할 수 있음
	닫는 태그가 없는 태그로 태그의 속성을 통해 정보를 제공함
	description, keywords, author, classification 등의 정보를 표기 가능하며 검색엔진은
	이런 정보를 활용함
	<head>
		<meta charset="utf-8">
	</head>

2) <body> 태그 내부에 들어가는 태그
- <br>
	html은 코드 가독성 향상을 위해 줄바꿈을 해도 줄바꿈이 화면에 출력되지 않음
	줄바꿈 태그로 텍스트 문단 중 원하는 위치에서 줄바꿈이 적용됨

- <p>
	paragraph 문단의 약자
	하나의 문단을 만들 때 사용하는 태그 <p>~</p>
	내용이 길면 웹 브러우저 창의 너비에 맞게 자동으로 줄바꿈이 됨

- <b>
	bold 굵게의 약자
	글자를 굵게 표시하는 태그 <b></b>

- <strong>
	<b>태그와 같은 기능을 하며 최신 표준은 <strong> 태그를 권고함
	중요한 텍스트를 굵게 표시하며 화면 낭독기에서 강조에서 읽어줌

- <i>
	italic 기울여서 쓰기의 약자
	글씨를 우측으로 비스듬히 기울여서 쓰기 <i></i>

- <em>
	중요한 텍스트를 기울게 표시 <em></em>
	화면 낭독기에서 강조해서 읽어줌	

- <h1>~<h6>
	섹션, 문단을 제목을 나타내며 숫자가 작을수록 글씨가 커짐 <h1></h1>~<h6></h6>

- <a>
	하이퍼링크를 걸어주는 태그 <a></a>
	속성	href : 클릭시 이동할 링크
		target : 링크를 여는 방법	
			_self : 현재페이지(기본값)
			_blank : 새탭
			_parent : 부모 페이지, iframe 등이 사용된 환경에서 쓰임
			_top : 최상위 페이지, iframe 등이 사용된 환경에서 쓰임
			프레임이름 : 직접 프레임 이름을 명시해서 사용할 수도 있음

- <img>
	이미지를 삽입하는 태그
	이미지 파일이 src 속성에서 지정한 경로에 없을경우 이미지는 출력되지 않거나 엑스박스가 뜨게 됨
	속성 	src : 이미지 경로
		width : 이미지 가로 크기
		height : 이미지 세로 크기

- 리스트 태그
	<ul> 
		unordered list의 약자, 순서없이 모양으로 목록을 만듦
	
	<ol>
		ordered list의 약자, 번호를 매겨 순서있는 목록을 만듦

	<li>
		list의 약자, 목록을 만드는 태그 <li></li>
		<ul></ul>, <ol></ol>태그 내부에서 사용됨

- 테이블 태그
	표를 만들기 위해서는 <tabel>태그를 사용
	로우(행, row)와 컬럼(열, column)을 표현하기 위해 <tr>, <td>, <th> 같은 태그와 함께 작성한다
	<tr> 행을 정의하는 태그
	<td> 각 행에서 열(칸)을 나누는 태그
	<th> 제목에 해당하는 열을 만드는 태그
	<thead>, <tbody>

	colspan	컬럼(열)을 확장할 때 사용
		좌우로 원하는 칸만큼 시작위치의 th 태그에 속성으로 지정한다
	rowspan 로우(행)을 확장할 때 사용
		위아래로 원하는 칸만큼 시작위치의 td 태그에 속성으로 지정한다
